# Wanas ونس

A personal quotes platform where users discover curated quotes matched to their current mood, powered by semantic search. Every quote is hand-picked from real books, attributed to its author, and surfaced through an AI-driven mood matching engine built on vector embeddings.

## Stack
- **Frontend:** React (Vite) — deployed on Vercel
- **Backend:** Django + Django REST Framework — deployed on Railway
- **Database:** PostgreSQL + pgvector on Railway
- **AI:** sentence-transformers for mood-to-quote semantic matching
